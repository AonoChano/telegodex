# Markdown 与富文本

Telegodex 使用 Telegram 原生富消息 API 渲染回复。这意味着 AI 输出的
Markdown——表格、代码块、公式——会以清爽样式呈现在对话中,而不是一串
转义的源码。

本章本身也通过同一管线渲染,因此下方每一个示例都是该语法在你对话中
实际呈现效果的实时预览。

---

## 语法速查

下表把每种元素的语法与渲染效果并排展示:左侧是源码写法,右侧是实际
渲染效果。

| 元素 | 语法 | 预览 |
|---|---|---|
| 标题 | `# 标题` | (即本章上方标题样式) |
| 粗体 | `**粗体**` | **粗体** |
| 斜体 | `*斜体*` | *斜体* |
| 行内代码 | `` `代码` `` | `代码` |
| 链接 | `[文本](url)` | [Telegodex](https://github.com/) |
| 引用 | `> 引用` | > 引用 |
| 行内 LaTeX | `$a^2+b^2$` | $a^2+b^2$ |

---

## 代码块

带围栏的代码块会保留格式与语言标签。实时预览:

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"
```

```powershell
Get-ChildItem -Path . -Filter *.py | Measure-Object
```

分享代码时请使用代码块——行内反引号仅用于短标识符。

---

## 表格

表格会渲染为真正的 Telegram 表格,列对齐。务必包含表头行,并在其后紧跟
`|---|` 分隔符。实时预览:

| 名称 | 分数 | 名次 |
|---|---|---|
| 甲 | 12 | 🥇 |
| 乙 | 9 | 🥈 |
| 丙 | 5 | 🥉 |

管道符与分隔行不可省略;缺少它们,这些行会被当作普通文本渲染。

---

## LaTeX

数学公式会渲染为 Telegram 原生公式,而非纯文本。行内公式用 `$...$`,
块级公式用 `$$...$$`。

行内预览:著名恒等式 $E = mc^2$ 在此处以真实公式呈现,而不是字面字符
E、=、m、c、2 拼接。

块级预览:

$$\int_0^1 x^2 \, dx = \frac{1}{3}$$

请勿用 Unicode 字符(如 ² 或 √)替代——交给 LaTeX 处理,才能在任意
字号下保持清晰渲染。

---

## 列表

无序列表与有序列表都原生渲染:

- 无序项一
- 无序项二
  - 嵌套子项
- 无序项三

1. 第一步
2. 第二步
3. 第三步

步骤、选项或任何序列都适合用列表——在 Telegram 中比逗号堆叠更整洁。
