# Markdown 与富文本

Telegodex 使用 Telegram 原生富消息 API 渲染回复。这意味着 AI 输出的
Markdown——表格、代码块、公式——会以清爽样式呈现在对话中,而不是一串
转义的源码。

---

## 支持的语法

| 元素 | 语法 |
|---|---|
| 标题 | `#`、`##`、`###` |
| 粗体 | `**文本**` |
| 斜体 | `*文本*` |
| 行内代码 | `` `代码` `` |
| 代码块 | <code>```lang ... ```</code> |
| 表格 | `\| 列 \| 列 \|` |
| 链接 | `[文本](url)` |
| 引用 | `> 引用` |
| LaTeX(行内) | `$a^2 + b^2$` |
| LaTeX(块级) | `$$E = mc^2$$` |

---

## 代码块

带围栏的代码块会保留格式与语言标签:

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"
```

分享代码时请使用代码块——行内反引号仅用于短标识符。

---

## 表格

表格会渲染为真正的 Telegram 表格,列对齐。务必包含表头行,并在其后紧跟
`|---|` 分隔符:

```markdown
| 名称 | 分数 |
|---|---|
| 甲   | 12 |
| 乙   | 9  |
```

---

## LaTeX

数学公式会渲染为 Telegram 原生公式,而非纯文本。行内公式用 `$...$`,块级
公式用 `$$...$$`。请勿用 Unicode 字符(如 ² 或 √)替代——交给 LaTeX
处理,才能在任意字号下保持清晰渲染。
