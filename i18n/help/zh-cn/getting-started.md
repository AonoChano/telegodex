# 快速开始

Telegodex 通过 Telegram 命令与聊天窗口底部的菜单按钮进行操作。本节列出
所有入口。

---

## 命令列表

| 命令 | 功能 |
|---|---|
| `/start` | 初始化机器人并查看状态 |
| `/new` | 开始新对话 |
| `/clear` | 清空当前对话 |
| `/settings` | 打开设置菜单 |
| `/help` | 打开本帮助手册 |
| `/language` | 切换界面语言 |
| `/model` | 切换模型或服务商 |
| `/history` | 查看对话历史 |

命令不区分大小写,可以在对话任意位置输入。

---

## 菜单按钮

输入框下方有四个常驻按钮:

- **💬 新对话** —— 等同 `/new`,开启新会话
- **📝 历史** —— 等同 `/history`,浏览过往消息
- **⚙️ 设置** —— 等同 `/settings`,打开设置菜单
- **ℹ️ 帮助** —— 等同 `/help`,打开本手册

不想输入命令时,直接点击按钮即可。

---

## 发送第一条消息

直接输入任意文字发送即可。Telegodex 会把消息路由到当前选中的服务商,并
流式回传回复。如果尚未配置任何服务商,机器人会提示你先打开 `/settings`。

每当话题切换、不希望旧上下文渗入时,使用 `/new` 即可。
